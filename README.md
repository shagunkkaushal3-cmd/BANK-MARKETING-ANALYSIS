# BANK MARKETING ANALYSIS
## Summary Objective
The main objective of this project is to analyze the given dataset to identify key trends, patterns, and business insights that support data-driven decision-making. The analysis aims to improve performance, optimize operations, and provide actionable recommendations based on meaningful data interpretation.
## Dataset Link
https://github.com/shagunkkaushal3-cmd/BANK-MARKETING-ANALYSIS/blob/main/1744721963590.zip
## KPI'S
1. What is the total revenue generated during the given time period?
2. Which product/category generates the highest sales?
3. What is the average order value (AOV)?
4. What percentage of orders were delivered on time?
5. Which region or warehouse has the highest number of delivery delays?
6. What is the inventory turnover ratio for each product category?
7. Which products frequently face stock-out situations?
8. What is the overall profit margin (%) of the business?
9. How does sales performance vary monthly or quarterly?
10. Which factors (price, demand, stock level, delivery time) are most affecting customer satisfaction or sales performance?
